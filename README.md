# KeenKeeper 

A friendship management web app that helps you stay intentional about the relationships that matter most. KeenKeeper lets you track when you last reached out to friends, log interactions, and get a clear picture of your social health — all in one place.

## Technologies Used

- Next.js - Full-stack React framework with App Router
- Tailwind-CSS - Styling and responsive design
- Recharts - Interactive charts for the analytics page

## Key Features

 ### Friend Dashboard with Status Tracking 
  View all your friends as cards showing their photo, tags, days since last contact, and a color-coded status (on-track, almost due, or overdue) so you always know who needs your attention
 ### Interaction Logging with Timeline
  From any friend's detail page, log a Call, Text, or Video check-in with one click — every interaction is automatically saved to a full timeline history you can filter by type
 ### Friendship Analytics & Goals
  Set personal contact goals for each friend, track your progress, and view a visual breakdown of your interaction patterns through a pie chart on the Stats page
